**Features**
------------

1.  **Data Extraction**:
    
    *   Extracts key fields such as **INLINE\_3D**, **CROSSLINE\_3D**, **SourceX**, **SourceY**, and **Scaler** using the centralized **DataProvider** class.
        
    *   Allows users to specify byte locations or header names dynamically for flexibility.
        
2.  **Trace Header Analysis**:
    
    *   Dynamically extracts trace header information (e.g., byte offsets, values) and saves it to an Excel file.
